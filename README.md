 # Sales Forecasting Project

## Overview
This project forecasts future sales based on historical sales data using Machine Learning regression techniques. It includes data preprocessing, model training, and prediction scripts.

## Features
- Predict future sales
- Data preprocessing and cleaning
- Model training and evaluation
- Beginner-friendly ML project

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Pickle
